## 1. 产品概述

旧衣改造搭配台是一款面向学生党和环保生活爱好者的可视化工具，帮助用户重新利用旧衣服，提供搭配灵感和改造思路。通过衣橱管理、改造模板、穿搭试排和场景推荐四大核心功能，让闲置衣物焕发新生。

- 解决问题：旧衣物丢弃浪费、缺乏搭配灵感、改造无从下手
- 目标用户：学生、环保生活爱好者、穿搭初学者
- 产品价值：降低服装浪费，提升穿搭创意，培养可持续消费习惯

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 录入衣橱、浏览改造模板、搭配试排、场景筛选 |

### 2.2 功能模块
1. **首页/衣橱页**：单品卡片展示、分类筛选、上传入口
2. **改造模板页**：裁剪、染色、拼接、装饰四大类改造方案
3. **搭配试排页**：拖拽式穿搭画布、单品选择区、保存功能
4. **场景推荐页**：上课、通勤、出游、拍照等场景穿搭组合

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 衣橱页 | 顶部导航 | 品牌logo、功能切换标签 |
| 衣橱页 | 单品卡片网格 | 展示已录入衣物，支持分类筛选、删除 |
| 衣橱页 | 上传区域 | 点击或拖拽上传衣物图片，填写名称、类别、颜色 |
| 改造模板页 | 分类标签 | 裁剪、染色、拼接、装饰四个分类切换 |
| 改造模板页 | 方案卡片 | 展示改造前后对比图、步骤说明、难度等级 |
| 搭配试排页 | 单品选择区 | 左侧展示所有可用衣物，按类别分组 |
| 搭配试排页 | 穿搭画布 | 中间拖拽区域，可放置上衣、下装、配饰 |
| 搭配试排页 | 操作栏 | 清空画布、保存搭配、随机搭配按钮 |
| 场景推荐页 | 场景标签 | 上课、通勤、出游、拍照、约会等场景 |
| 场景推荐页 | 推荐组合 | 展示对应场景的穿搭组合卡片 |

## 3. 核心流程

用户打开应用 → 录入衣橱单品（上传图片+填写信息）→ 浏览改造模板获取灵感 → 在搭配试排区拖拽组合衣物 → 根据场景筛选推荐穿搭 → 保存满意的搭配方案

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：大地绿 #5B8C5A（环保自然）、米杏色 #F5F0E6（温暖复古）
- **辅助色**：陶土橙 #D4846F（创意活力）、深棕 #4A3F35（稳重质感）
- **按钮风格**：圆角矩形，轻微阴影，hover时有缩放效果
- **字体**：标题用 Noto Serif SC（优雅衬线），正文用 Noto Sans SC（清晰易读）
- **布局风格**：卡片式布局，柔和阴影，大量留白
- **图标风格**：线性图标，统一2px描边，圆角处理

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 衣橱页 | 单品卡片 | 圆角图片、标签式分类、悬停浮起效果 |
| 改造模板页 | 方案卡片 | 前后对比图、步骤序号、难度星级 |
| 搭配试排页 | 拖拽画布 | 虚线网格背景、拖拽时半透明预览 |
| 场景推荐页 | 场景标签 | 胶囊形状、选中时填充主色 |

### 4.3 响应式
- Desktop-first 设计，1280px 起步
- 平板端：侧边栏收起为图标模式
- 移动端：底部标签导航，卡片单列展示，触摸优化

### 4.4 动效设计
- 页面切换：淡入淡出 + 轻微位移
- 卡片悬停：向上浮动 4px + 阴影加深
- 拖拽操作：半透明跟随 + 放置动画
- 按钮交互：scale(0.98) 按压效果
